# Single Page CV

**Project Link:** [https://roadmap.sh/projects/single-page-cv](https://roadmap.sh/projects/single-page-cv)

## Project Overview

This project focuses on creating a professional single-page CV using semantic HTML. The goal is to build a well-structured, SEO-optimized, and socially shareable resume that serves as a foundation for future styling and enhancements.

## Key Requirements

- **Semantic HTML:** Use appropriate HTML tags to structure your CV.
- **SEO Meta Tags:** Include essential meta tags for search engine optimization.
- **Open Graph (OG) Tags:** Add OG tags for better social media sharing.
- **Favicon:** Add a favicon for your CV page.
- **Structure:** The CV should be easily understandable and ready for styling in future projects.

## Submission Checklist

- [x] Semantically correct HTML structure.
- [x] Single-page layout with sections for education, skills, and career history.
- [x] SEO meta tags in the head section.
- [x] OG tags for better social media sharing.
- [x] A favicon linked in the head section.

## Current Status

The basic HTML structure is complete with all required elements implemented. The CV includes personal information, skills, education, and experience sections.

## Getting Started

1. Clone this repository
2. Open `Single-Page CV/index.html` in your browser
3. Start implementing the future tasks listed above
